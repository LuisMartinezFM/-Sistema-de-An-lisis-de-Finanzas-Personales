# Portada y Navegación

## Vista de la portada

![Portada](../imagenes/portada.png)

La portada funciona como **menú principal del sistema**, permitiendo acceder de forma directa a cada uno de los capítulos del análisis financiero.

## Estructura de navegación

El proyecto cuenta con dos mecanismos de navegación complementarios:

### Menú principal
Desde la portada es posible acceder directamente a:
- Resumen general
- Ingresos y Uso de Liquidez
- Ahorro y Metas
- Préstamos

### Navegación secuencial
Todas las pantallas incluyen controles de navegación:
- **Previous**: pantalla anterior
- **Home**: regreso al menú principal
- **Next**: pantalla siguiente

Este diseño permite una exploración libre o secuencial del sistema, según el interés del usuario.
